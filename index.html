WJ Weather Journal

A Flask-based web app tailored for anyone interested in logging and tracking weather conditions. Its mobile-friendly design ensures you can document weather patterns anytime, anywhere. Developed as a final project for Hardvard's CS50x Introduction to Computer Science course

Key Features:

  • User Authentication: Secure login system.
  • Flexible Weather Logging: Enter details across categories like air, water, and more, with automatic timestamps.
  • Data Integrity and Viewing: Backend validation ensures accurate records. View historical data with ease.
  • Responsive Design: Optimized for both mobile and desktop use, with intuitive sliders and dropdowns.
  • Custom Notes and Moon Phases: Add personal observations and enjoy visual moon phase indicators.

Tech Stack:

  • Frontend: HTML, CSS, and JavaScript.
  • Backend: Python and Flask.
  • Database: SQLite.
  • Security: Werkzeug and Flask sessions.
